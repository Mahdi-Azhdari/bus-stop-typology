# MBTA Bus Stop Typology Analysis

This repository contains the code and Jupyter notebooks used to develop a stop-level typology for the MBTA bus network using Gaussian Mixture Models (GMM) and interpretable machine-learning methods such as Random Forests, SHAP, and XGBoost. The workflow includes data preprocessing, clustering, model explanation, and model validation.

## Directory Structure
- `Bus_stops_typology/`: The Jupyter notebook used for the analysis.
- `LICENSE`: MIT license information.
- `README.md`: Project overview.

## How to Use
1. Clone this repository.
2. Install the necessary Python packages listed in `requirements.txt`.
3. Run the Jupyter notebooks in the directory to reproduce the analysis.

## Data
The AFC/APC datasets and GIS layers used in this study are **not included** due to non-disclosure agreements with the MBTA.  
The notebooks describe the expected data structure so the workflow can be adapted to similar datasets.
